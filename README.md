## Setting Up Locally

To set up Taxbot locally, follow these steps:

1. Clone the repository

2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

- open (http://localhost:5173/)
